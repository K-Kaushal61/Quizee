# Quizee
A modern, interactive quiz application built with React, TypeScript, Tailwind CSS, and Framer Motion.  
The app focuses on smooth animations, clean UI, and an engaging user experience.

# Tech Stack Used

• React – Component-based UI development

• TypeScript – Type safety and better developer experience

• Vite – Fast build tool and development server

• Tailwind CSS – Utility-first styling

• Framer Motion – Smooth animations and transitions

# Key Features Implemented

1.Animated Progress Bar

• Shows partially filled state for unanswered questions

• Fully fills when a question is answered

2.Interactive Question Cards

• Smooth transitions between questions

3.Navigation Controls

• Back and Next navigation between questions

4.Animated Final Score

• Score loads using a vertical rolling animation

• Final score locks smoothly in place

5.Restart Quiz

• Allows users to retake the quiz from the beginning

6.Modern UI

• Gradient backgrounds

• Rounded containers

• Clean spacing and visual hierarchy

# Assumptions Made

• The application is optimized for desktop screens

• Quiz questions are loaded from a static data source

• Only one correct answer exists per question
